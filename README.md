# Portfolio

Astro-based personal portfolio.

The top page is a WebGL fluid-style link portal:

- Personal Page: `/profile/`
- GitHub: `https://github.com/shobioinfo`
- Scholar: `https://scholar.google.co.jp/citations?view_op=list_works&hl=ja&hl=ja&user=QwiO-48AAAAJ`
- LinkedIn: `https://jp.linkedin.com/in/sho-oshima-508316309`
- Qiita: `https://qiita.com/shobioinfo`

## Develop

```bash
npm install
npm run dev
npm run build
npm run preview
```

## GitHub Pages

This project is configured for:

```text
https://shobioinfo.github.io/sho_oshima.github.io/
```

`astro.config.mjs` sets `site: 'https://shobioinfo.github.io'` and
`base: '/sho_oshima.github.io'`.
